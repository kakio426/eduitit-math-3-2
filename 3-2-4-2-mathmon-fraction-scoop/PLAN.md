# 매스몬 분수만큼 담기 제작 계획 (3-2-4-2)

## Context (왜 만드나)
- 단원/차시: 3학년 2학기 4단원 2차시, 영역 수와 연산, 성취기준 `[4수01-09]`(안내용 — 원문 대조 필요).
- 배움주제: 전체의 분수만큼 구하기. 4-1(부분/전체) 다음, 분수를 양으로 다루는 차시.
- 한 줄 학습 목표(학생용): 전체를 똑같이 나눠 분수만큼 담는 개수를 구한다.

## 수학 설계
- 문제 유형: 전체 `N`개의 `num/den`만큼 = `N÷den×num`. 예 12개의 3/4 = 9.
- 생성 규칙: `N ∈ {6,8,9,10,12,15,16}`, `den`은 `N`의 약수(2..N-1, `N/den≥2`), `num ∈ [1, den-1]`(진분수). 한 판 10문제.
- 메타 필드: `total(N)`, `den`, `num`, `perGroup(N/den)`, `answer(perGroup×num)`.
- 대표 오개념: 한 묶음만 답(`N/den`), 분모 개수만큼(`den`), 곱셈 생략.

## 단계 선택 설계 (핵심 메커니즘 — 2단계)
- 1단계 "한 묶음 크기": 정답 `N/den`. 대표 오답: 분모 `den`, 전체 `N`.
- 2단계 "담을 개수": 정답 `perGroup×num`. 대표 오답: 한 묶음만(`perGroup`), 분모(`den`).
- 1단계 정답이 그룹 토큰 보드에 반영되고, 2단계에서 담는 `num`묶음을 강조 → 최종 답 자동 완성.

## 보상/랜덤 엔진 (라벨만 교체)
- 증가=가득 담기 · 감소=흘림 · 0=빈 바구니 · 즉시완성=정확히 담기 · 특별=전설 바구니(무지개).

## 효과 설계
- 단계 정답 직후: 보드 pop/shake, 담는 묶음 토큰이 민트로 반짝.
- 2단계 진입 시 담는 묶음 강조(`group-box.is-on`).
- 결과 전 측정 + rAF 보상 팝 + `prefers-reduced-motion` 대응.

## 결과 등급
