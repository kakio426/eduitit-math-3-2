# 곱셈 상자런

에듀잇티 수학 게임 시리즈 1탄입니다.

- 대상: 3학년 2학기 1단원
- 학습: 받아올림 없는 세 자리 수 × 한 자리 수
- 방식: 문제를 맞히고 상자를 열어 점수를 크게 바꾸는 짧은 반복 게임
- 실행: `index.html`을 브라우저에서 열기
