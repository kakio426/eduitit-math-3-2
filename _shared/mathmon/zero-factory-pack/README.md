# 0 공장 매스몬팩

3차시 `매스몬 0 공장` 결과 카드용 신규 매스몬 10종입니다.

## 생성 방향

- 기존 기본 매스몬팩의 알, 병아리, 여우, 용, 사자, 독수리, 티라노, 유니콘, 킹드래곤 계열과 겹치지 않게 제작했습니다.
- 3차시 학습 소재인 `0 처리`와 게임 테마인 `0 공장/컨베이어`에 맞춰 공장 기계, 부품, 검사, 포장 실루엣으로 구성했습니다.
- 모든 캐릭터는 카드 합성에 쓰기 쉽도록 투명 배경 PNG와 WebP를 함께 보관합니다.

## 사용처

- 원본 관리: `_shared/mathmon/zero-factory-pack/`
- 3차시 배포 복사본: `3-2-1-3-mathmon-zero-factory/assets/mathmon/zero-factory-pack/`
- 3차시 `index.html`의 `MATHMONS` 배열에서 결과 카드 보상으로 사용합니다.
