# 매스몬 피자 분수 가게 제작 계획 (3-2-4-1)

## Context (왜 만드나)
- 단원/차시: 3학년 2학기 4단원 1차시, 영역 수와 연산, 성취기준 `[4수01-09]`(안내용 — 교육부 고시 제2022-33호 원문 대조 필요).
- 배움주제: 부분과 전체로 분수 나타내기. 단원 도입(기초) 차시.
- 한 줄 학습 목표(학생용): 전체 조각 중 색칠된 조각을 분수로 나타낸다.

## 수학 설계
- 문제 유형: 원(피자)을 `den`등분, `num`조각 색칠 → 분수 `num/den` 고르기.
- 생성 규칙: `den ∈ {2,3,4,5,6,8}`, `num = randomInt(1, den-1)`. 한 판 10문제.
- 메타 필드: `num`(색칠=분자), `den`(전체=분모), `correctKey`.
- 대표 오개념(보기에 항상): 분자·분모 뒤바꿈 `den/num`, 색칠 안 된 조각으로 분자 `(den-num)/den`.

## 단계 선택 설계 (핵심 메커니즘)
- 1단계: 피자 SVG를 보고 분수 4개 중 정답 `num/den` 선택.
  - 정답: `num/den`. 대표 오답: 뒤바꿈(`den/num`)·여집합(`(den-num)/den`) 중 하나 이상 항상 포함 + 전체±1/색칠±1 보충.
- 마지막(유일) 단계 정답 → 보상 룰렛 → 문제 완료.

## 보상/랜덤 엔진 (라벨만 교체)
- 증가=토핑 추가 · 감소=한 입 먹힘 · 0=바닥 · 즉시완성=완벽한 피자 · 특별(최고 직행)=전설 피자(무지개).
- 한 문제에서 한 번이라도 틀리면 누수(감소) 처리 → 일부러 틀리기 차단.

## 효과 설계
- 단계 정답 직후: 피자 보드 `board-pop`(정답)·`board-shake`(오답), 색칠 조각이 민트로 반짝.
- 문제 완료 후: 피자 점수 미터 이동 + 보상 팝 rAF 모션.
- 랜덤 이벤트 충격: 증가/대량/감소/0/완벽/전설을 글 없이 art 클래스로 구분.
- 결과 전 측정: 모은 피자 점수를 스캔/측정 후 등급 공개.
- `prefers-reduced-motion`에서 보드 애니메이션 정지.

## 결과 등급 (차시 자체 완결형 — 도감 아님)
- 한 조각 → 반 판 → 한 판 → 특대 판 → 가게 대박 + 무지개(전설 피자), 정답 수 게이트.
- "이번엔 한 판! 다음엔 가게 대박!"
