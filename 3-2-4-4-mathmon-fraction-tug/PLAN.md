# 매스몬 분수 줄다리기 제작 계획 (3-2-4-4) ★ 단원 정점

## Context (왜 만드나)
- 단원/차시: 3학년 2학기 4단원 4차시, 영역 수와 연산, 성취기준 `[4수01-10]`(안내용 — 원문 대조 필요).
- 배움주제: 분모가 같은 분수와 단위분수의 크기 비교. **4단원 정점** — 부분/전체(4-1)·분수만큼(4-2)·분수 종류(4-3)에서 키운 분수 감각을 "두 분수 크기 비교"로 종합.
- 한 줄 학습 목표(학생용): 두 분수 중 더 큰 분수를 고른다.

## 수학 설계
- 문제 유형: 두 분수 비교, 한 문제 한 유형. 한 판 10문제.
  - (A) 분모 같음: 같은 `den ∈ [3,8]`, 서로 다른 `num` → 분자 큰 쪽이 큼. 예 `3/7` vs `5/7` → 5/7.
  - (B) 단위분수: `num=1`, 서로 다른 `den ∈ [2,8]` → 분모 작은 쪽이 큼. 예 `1/3` vs `1/5` → 1/3.
- 대표 오개념: 단위분수에서 분모 큰 쪽을 크다고(`1/5>1/3`), 분모 같을 때 분모로 비교, 분자만/분모만 보고 성급 판단.

## 단계 선택 설계 (핵심 메커니즘)
- 1단계 **2지선다**: 두 분수 중 더 큰 쪽 선택. 정답 위치는 좌/우 랜덤.
- 보드에 두 분수를 **같은 길이 막대에 칸을 채워** 보여 줘, 같은 분모는 칸 수, 단위분수는 한 칸 크기로 길이를 직접 비교.

## 보상/랜덤 엔진 (라벨만 교체)

## 효과 설계
