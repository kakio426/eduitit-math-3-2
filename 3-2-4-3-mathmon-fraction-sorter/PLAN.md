# 매스몬 분수 분류 컨베이어 제작 계획 (3-2-4-3)

## Context (왜 만드나)
- 단원/차시: 3학년 2학기 4단원 3차시, 영역 수와 연산, 성취기준 `[4수01-10]`(안내용 — 원문 대조 필요).
- 배움주제: 진분수·가분수·대분수 구분. 분수의 종류를 이름과 모양으로 익히는 차시.
- 한 줄 학습 목표(학생용): 나온 분수를 진분수·가분수·대분수 통으로 분류한다.

## 수학 설계
- 문제 유형: 분수 1개 제시 → 진분수/가분수/대분수 통 3개 중 선택. 한 판 10문제.
- 생성 규칙:
  - 진분수: `den ∈ [3,8]`, `num ∈ [1, den-1]`(분자<분모).
  - 가분수: `den ∈ [2,5]`, `num ∈ [den, 2den-1]`(분자≥분모, `5/5` 포함, 정수배 제외).
  - 대분수: `whole ∈ [1,3]`, `den ∈ [2,5]`, `num ∈ [1, den-1]`(자연수+진분수).
- 대표 오개념: 가분수↔대분수 혼동, 분자=분모(`5/5`)를 진분수로 착각, 대분수를 가분수로.

## 단계 선택 설계 (핵심 메커니즘)
- 1단계(분류): 분수 보고 통 3개 중 하나 선택. 정답=실제 종류.
- 보드에 분수를 **숫자(분수/대분수 표기) + 원 조각 그림**으로 동시에 보여 줘 분자·분모·자연수 관계를 눈으로 비교.

## 보상/랜덤 엔진 (라벨만 교체)
- 증가=정확 분류 · 감소=오분류 · 0=라인 막힘 · 즉시완성=완벽 분류 · 특별=전설 라인(무지개).

## 효과 설계
- 단계 정답 직후: 보드 pop/shake, 색칠 조각 민트 반짝.
- 결과 전 측정 + rAF 보상 팝 + `prefers-reduced-motion` 대응.

## 결과 등급
- 첫 분류 → 줄 분류 → 라인 → 큰 라인 → 공장장 + 무지개(전설 라인), 정답 수 게이트.
- "이번엔 라인! 다음엔 공장장!"

## 화면 (5개)
- 문제 화면 과밀 방지: 큰 분수(숫자+원)·한 줄 지시문·통 3선택지만. 세 종류 정의 표를 펼치지 않음.
