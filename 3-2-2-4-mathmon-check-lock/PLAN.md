# 매스몬 검산 자물쇠 제작 계획 (3-2-2-4) ★ 단원 정점

## Context (왜 만드나)
- 2단원 나눗셈 4차시 · 영역: 수와 연산 · 성취기준 `[4수01-06]`(안내용 — 원문 대조 필요)
- 배움주제: **곱셈으로 검산하기** — `몫×나누는수+나머지 = 나누어지는 수`
- 단원 위치: **2단원 종합**. 나눗셈(1~3차시) + 1단원 곱셈을 한 흐름에서 사용
- 학생용 한 줄 목표: "검산으로 자물쇠를 풀어 금고를 열자!"

## 수학 설계
- 문제 유형: 나눗셈 결과(몫·나머지)를 주고 `몫×나누는수+나머지`로 원래 수 복원/확인 (예 `9×5+2=47`)
- 생성 규칙: 3차시와 같은 방식으로 dividend/divisor/quotient/remainder 생성 → 검산식 구성. 한 판 10문제 랜덤. 곱은 1단원 범위(한·두 자리) 안으로.
- 메타 필드: `quotient`, `divisor`, `remainder`, `product`(몫×나누는수), `answer`(=product+remainder=dividend)
- 대표 오개념(고정 오답): **나머지 더하기 누락**(몫×나누는수만으로 자물쇠 시도)

## 단계 선택 설계 (핵심 메커니즘)
- **1단계 곱하기**: `몫 × 나누는수` 계산 (예 9×5=45) — 1단원 곱셈 소환
- **2단계 나머지 더하기**: `곱 + 나머지` (예 45+2=47) — 받아올림 덧셈
- **3단계 자물쇠 맞추기**: 결과가 원래 수와 일치하는지 다이얼 맞추기 → 금고 열림 자동 연출
- 곱셈+덧셈을 한 문제에서 종합(1단원 4차시 "합체"와 같은 종합형 구조)

## 보상/랜덤 엔진 (라벨만 교체 — 보안실/금고)
- 증가(+/×): **보안 강화** · 감소(−/÷): **오류** · 0: **잠김** · 즉시완성: **마스터키(조기 마무리)** · 특별: **무지개 코어(최고 직행)**
- 한 문제 중간 오답 → **오류** 처리(일부러 틀리기 차단)

## 결과 등급 (차시 자체 완결형 — 도감 아님)
- 금고 등급: **자물쇠 → 금고 → 대형금고 → 비밀금고 → 보물고** + **무지개(전설 금고)** , 정답 수 게이트
- "다음엔 더 멀리": "이번엔 비밀금고! 다음엔 보물고까지!"

## 화면 (5개 + 선택)
1. 첫 화면: 제목 `매스몬 검산 자물쇠`, 한 줄 목표, 시작 버튼, 금고/보안실 + 매스몬
2. 설명: 곱하기 → 나머지 더하기 → 자물쇠 맞추기
3. 문제: 검산 단계 선택판(다이얼) 최우선
4. 보상: 자물쇠 풀리고 금고 열리는 연출 + 랜덤 보안 결과
5. 결과: 금고 등급 이미지 + 칭찬 + 정답 수
- (선택) 내 금고 카드 받기
