# 매스몬 별 줍기 제작 계획 (3-2-2-3)

## Context (왜 만드나)
- 2단원 나눗셈 3차시 · 영역: 수와 연산 · 성취기준 `[4수01-06]`(안내용 — 원문 대조 필요)
- 배움주제: **나머지가 있는 나눗셈** — 똑같이 나누고 남는 나머지의 의미, "나머지 < 나누는 수"
- 단원 위치: 나눗셈에 "나머지" 개념 도입(검산 4차시의 토대)
- 학생용 한 줄 목표: "똑같이 나누고 남는 별을 주워 보자!"

## 수학 설계
- 문제 유형: `(몇십몇) ÷ (몇) = 몫 … 나머지` (예 `47÷5=9…2`, `38÷4=9…2`, `29÷6=4…5`)
- 생성 규칙: divisor d=3~9, 몫 q 무작위, 나머지 r=1~(d-1) 무작위 → `dividend=q*d+r`. 나머지 0 제외(반드시 나머지 발생). 한 판 10문제 랜덤.
- 메타 필드: `dividend`, `divisor`, `quotient`, `remainder`
- 대표 오개념(고정 오답): **나머지 ≥ 나누는 수**(예 47÷5을 8…7로) — 더 나눌 수 있는데 멈춤

## 단계 선택 설계 (핵심 메커니즘)
- **1단계 몫 고르기**: dividend 안에 divisor가 몇 번 들어가는지 (예 47에 5가 9번)
- **2단계 나머지 고르기**: 남는 수 (예 47-45=2), 보기에 "나누는 수보다 큰 값" 오답 포함
- **3단계 검산 미리보기**: `몫×나누는수+나머지 = dividend` 자동 확인(별이 다 모이는 연출) — 4차시 검산 예고
- 나머지 자체가 "주운 별"로 시각화

## 보상/랜덤 엔진 (라벨만 교체 — 밤하늘)
- 증가(+/×): **유성우** · 감소(−/÷): **구름** · 0: **깜깜(빛 가림)** · 즉시완성: **별똥별(조기 마무리)** · 특별: **무지개 별(최고 직행)**
- 한 문제 중간 오답 → **구름** 처리(일부러 틀리기 차단)
- ③ 특화: 등급 산정에 **나머지를 보너스 별로 가산** → 점수가 낮아도 "남는 것"이 가치가 됨(못해도 주울 게 있음)

## 결과 등급 (차시 자체 완결형 — 도감 아님)
- 줍은 별 등급: **별먼지 → 작은별 → 별자리 → 은하 → 대우주** + **무지개(초신성)** , 정답 수 게이트
- "다음엔 더 멀리": "이번엔 별자리! 다음엔 은하까지!"

## 화면 (5개 + 선택)
1. 첫 화면: 제목 `매스몬 별 줍기`, 한 줄 목표, 시작 버튼, 밤하늘 + 매스몬
2. 설명: 몫 고르기 → 나머지 고르기 → 검산으로 별 모으기
3. 문제: 몫·나머지 단계 선택판 최우선
4. 보상: 나머지 별 줍기 + 랜덤 밤하늘 결과
5. 결과: 별 등급 이미지 + 칭찬 + 정답 수
- (선택) 내 별 등급 카드 받기
